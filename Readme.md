
Use the following commands to run the tasks:

python main.py – runs Part 1 and Part 2 tasks with default hyperparameters.

python main.py part1 – runs Part 1 (Encoder + Classifier for speech classification).

python main.py part2 – runs Part 2 (Decoder for language modeling).

python main.py part3 – runs Part 3 experiment comparing standard positional embeddings and ALiBi.
